# Databricks with PySpark

Azure Databricks development environment with PySpark, Databricks CLI, Asset Bundles, and pre-commit tools

## Usage

See the [main repository documentation](../../README.md) for instructions on using this template.

Reference this template in VS Code:
```
ghcr.io/arclytics/arc-devcontainers/databricks
```

## What's Included

- Python 3.11 (configurable)
- Databricks CLI with Asset Bundles support
- PySpark, pandas, numpy, pyarrow
- Pre-commit hooks with Ruff, MyPy
- Azure CLI
- Comprehensive Python development tools
